    FAQ

    Who is this for?

    The program is targeted towards middle school students, although elementary/high school students are welcome to attend!

    Is there any cost associated?

    No, Code One is a 100% free program designed to increase computer science exposure to anyone and everyone, even those who may lack the proper resources. All expenses are covered by Code One.

    Do I need to bring anything?

    Nope. All necessary materials will be provided by us; however, if you have a computer/laptop that you're comfortable with bringing, you're more than welcome!

    Do I need any prior experience?

    Nope. We welcome all experience levels! Our program is designed to start at the basics and work our way up to the more advanced stuff.

    What will I learn?

    A ton of stuff! Specifically, we will start out with Scratch and then work our way up to Python. More on this under Curriculum.

    Why should I learn to code?

    As Steve Jobs once said, "I think everybody in this country should learn how to program a computer because it teaches you how to think." We here at Code One agree. Plus, it's also a ton of fun!

    CURRICULUM

    Scratch

    Scratch is a simple, drag-and-drop programming language oriented towards game development for beginners. Its easy-to-use interface allows our students to pick up the basics of programming really quickly. We use it to teach problem deconstruction, computer logic, and basic algorithm design skills.

    Some topics we're going to cover:

    • Game development
    • Computer logic
    • Variables

    Find out more about Scratch here.

    Python

    Python is a powerful, real world, programming language used in many professions. However, its syntax (how the code is written) is super simple. This allows new students to pick up techniques much more quickly. We use Python to build upon what was learned with Scratch, but give them greater control over their projects.

    Some topics we're going to cover:

    • If-else statements
    • Loops
    • Variables
    • Functions
    • Algorithm design

    Find out more about Python here.

    Who are we?

    We are a group of high school students at North Allegheny who enjoy programming. However, we realize that not everyone has the resources that we do to learn how to program, so we decided to fix that by offering computer science classes at various locations around our community.
